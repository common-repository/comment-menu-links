Adds sub menu items to the Comments menu for each of the different comment statuses.

== Description ==

This is a very simple plugin that adds links for each of the different comment statuses to the Comments menu. This includes links for:

* Approved
* Pending
* Spam
* Trash

== Installation ==

1. Activate the plugin
2. Your "Comments" menu will now have sub menu items for each Comment Status
